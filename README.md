# FoodPlan

Сервис по подписке на персонализированные меню и рецепты. Приложение позволяет пользователям выбирать типы меню (например, классическое, кето, вегетарианское), исключать блюда с определенными аллергенами и оформлять платную подписку на нужное количество человек и месяцев.

## Основные возможности

* **Умная фильтрация рецептов:** Сервис автоматически исключает из выдачи пользователя рецепты, содержащие ингредиенты, на которые у него аллергия.
* **Гибкий конструктор подписки:** Динамический расчет стоимости в зависимости от срока (1, 3, 6, 12 месяцев), количества персон и выбранных приемов пищи (завтрак, обед, ужин, десерт).
* **Система промокодов:** Поддержка скидочных купонов с валидацией по сроку действия и автоматическим пересчетом итоговой цены (AJAX).
* **Кастомная аутентификация:** Вход в систему осуществляется по Email, а не по стандартному username.
* **Удобная админ-панель:** Кастомизированная панель администратора с предпросмотром изображений блюд, удобным управлением подписками пользователей и составом рецептов (Inline-модели).
* **Скрипт импорта данных:** Готовый инструмент для быстрой загрузки базы рецептов, ингредиентов и тегов из JSON-файла.

---

## Технологический стек

* **Backend:** Python, Django 4.2
* **База данных:** SQLite
* **Работа с изображениями:** Pillow 11.3.0
* **Frontend-интеграция:** HTML/CSS шаблоны Django, AJAX (JavaScript) для расчета цен.

## Установка и локальный запуск

1. Настройка виртуального окружения и зависимостей

```bash
python -m venv venv
source venv/bin/activate  # Для Linux/macOS
venv\Scripts\activate     # Для Windows
pip install -r requirements.txt
```

2. Подготовка базы данных

```bash
python manage.py makemigrations
python manage.py migrate
```

3. Создание суперпользователя

Для доступа к панели администратора создайте учетную запись:

```bash
python manage.py createsuperuser
```

4. Запуск сервера
```bash
python manage.py runserver
```

Сервер будет доступен по адресу: http://127.0.0.1:8000/
Панель администратора: http://127.0.0.1:8000/admin/
